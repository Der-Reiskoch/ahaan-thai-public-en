Developer Tools

Application Programming Interfaces

Thai Food Encyclopedia API

This interface provides the complete Thai Food Encyclopedia in JSON format. It contains structured data on hundreds of Thai dishes with detailed information such as a description of the dish, German and English translations, Thai names, regional categorisations and links to recipes.

Thai Food Dictionary API

This API provides the Ahaan Thai Food Dictionary in JSON format. It contains culiary Thai terms and divides these into different categories (e.g. spices, vegetables, meat, preparation methods). Each entry contains the word in Thai, German and English translations as well as romanisation in German and English spelling. The dictionary makes it possible to systematically look up Thai kitchen terms and translate between Thai, German and English.

Thai Cook Book Library API

This API provides my collection Thai Cookbooks and the recipes it contains in JSON format. Each cookbook entry contains two sections: info with meta information about the book (author, description, publisher, ISBN, year of publication, level of difficulty as well as image and shopping links) and recipes with all recipes from the book that i added to ahaan-thai.de. The library contains information for each recipe such as page numbers, assignment to a region, Thai script, German and English titles and romanisations. It also contains image paths and URLs for the individual recipes.

To make these interfaces available for AI tools, there is the MCP Bundle ahaan-thai.mcpb. You can download it and double-click to integrate it into any AI tool that understands MCPB, such as Claude Desktop.
